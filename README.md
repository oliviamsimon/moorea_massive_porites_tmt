## Overview
This is a project repository for Simon et al. Quantitative proteomics reveals a possible mechanism by which Porites spp. achieves homeostatic calcification under acidified conditions. 

## Layout

The repository contains three main files. Each file is briefly described below. 

### **`data.md`** 

* Final protein abundance dataset used for analysis.

### **`analysis.txt`** 
* Major functions and processing code used for analysis.

### **`workflows.md`** 
* Proteome Discoverer 3.1 workflows
